principais formatações de textos

negrito / destaque

o negrito é feito usando a tag 'b' ou a tag semantica 'strong'

italico / enfase

o italico é feito com a tag 'i', ou usando a tag semantica de enfase 'em'

o ideal é utilizar a tag semantica 'strong' e 'em' em HTML e as tags de forma b e i em CSS

texto marcado

a marcação de texto é feito com a tag 'mark'

texto deletado

podemos marcar um texto como excluido, para indicar que deve ser lido mas não considerado com a tag 'del'

texto inserido

podemos marcar um texto como inserido para indicar que ele foi adicionado posteriormente, com a tag 'ins'

texto sobrescrito

para inserir como x20+3, com a tag 'sup'

texto subscrito

para inserir como H2O, com a tag 'sub'

códigos fontes

o comando documento.getElementById('teste') é escrito em JavasCript, os caracteres ocupam o mesmo espaço com a tag 'code'

para copiar um codigo exatamente como foi escrito, incluindo as identações, se usa a tag 'pre'

LINKS

LINKS EXTERNOS

criando um link externo para o RYM, com a tag 'a', o parametro 'target'= _blank define o alvo para uma janela em branco, e 'rel = external' leva o link para fora do site.

LINKS INTERNOS

link para abrir a segunda pagina do site, com ESTILOS EXTERNOS, o arquivo referenciado precisa estar na mesma pasta. 'rel = next' referencia o link como 'proxima' pagina. 'target = self' referencia a propria pagina.

caso eu crie uma pagina em outra pasta, preciso informar o local para ancorar, e para voltar para a pagina anterior, o camando '../' antes do nome do arquivo funciona para referenciar que estamos ancorando a uma pagina anterior.

IMAGENS DINAMICAS

as imagens servem para diferentes dispositivos de tamanhos diferentes, através das tags 'source media type' e 'picture'.

a tag img é o padrao, caso caia na condição da tag 'source', ela passara a ser executada, no caso, caso seja menor do que 1000px, é carregado a imagem M, menor que 700px é carregado a P, e maior q isso a imagem padrão G. A ordem dos 'source' importam, tem q ser do menor para o maior.

AUDIOS

para reprodução de audios é usada as tags 'audio', e parametros 'controls'

suporta MP3, WAV E OGG

utilizando a tag 'audio', e a tag 'source' da msm forma das imagens dinamicas, o que é colocado é source é reproduzido caso necessario em ordem.

VIDEOS

colocando arquivos MP4 usando a tag mais simples 'video' e parametro 'width' e 'controls'(é recomendavel upar utilizando o source assim como nas outras midias e colocar diversos formatos(MP4,MKV,WEBV) para ter certeza que o site sera compativel com diversos navegadores e dispositivos.)

utilizando a tag 'video' e 'source' (recomendado, assim como nas outras midias). O parametro 'poster' coloca uma thumbnail no video. O problema deste método é a quandidade de banda consumida para hospedar os arquivos de video no servidor LOCAL, acaba saindo muito caro.

utilizando video do próprio youtube. Basta ir no video e clicar em compartilhar e em seguida 'incorporar' que gera um código HTML.
